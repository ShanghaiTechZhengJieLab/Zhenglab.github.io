CPLDhat --

two improved methods on recombination rate prediciton!

CLDhat method & PLDhat method :


Meiotic recombination is an essential event during the process of meiosis resulting in an exchange of segments between paired chromosomes. Measuring of recombination rate is crucial for understanding evolutionary inference.

We provide two programs: CLDhat method and PLDhat method. The CLDhat is designed based on LDhat implementing MCMC convergence diagnostic algorithms to automatically predict parameters and monitor the mixing process. The PLDhat employs parallel computation method to further accelerate the CLDhat program. They provide efficient and faster methods for recombination rate prediction.

CPLDhat is an open source Java program that predicts recombination rate dynamically and fast. The stand-alone package is freely available for download at Download. It has been tested on Windows XP/7 and Linux OS.
