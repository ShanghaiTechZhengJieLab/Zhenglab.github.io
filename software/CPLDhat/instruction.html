CPLDhat

Two methods are provided : CLDhat method and PLDhat method.

CLDhat method controls the process of rjMCMC iteration to supervise the Markov chain convergent status in order to accelerate mixing process and ensure the accuracy. PLDhat method allocates separate task to individual processor running single chain in parallel.

Note: PLDhat method is not integrated into the GUI program. A seperate shell script is provided to run PLDhat in a batch cluster environment.
